## 2. 变更概述

该提交意图将 `ao ui-browser-smoke` 从面向用户的 CLI 中移除，并删除 Playwright 依赖。

| 类别 | 变更 |
| --- | --- |
| 删除代码 | `src/ui/browser-smoke.ts`、`src/cli/commands/ui-browser-smoke.ts`、`test/ui-browser-smoke.test.ts` |
| CLI | 移除路由、help 文案 |
| 公共 API | 从 `src/index.ts`、`src/internal.ts` 移除 `runUiBrowserSmoke` 及相关类型 |
| CI | 删除 `smoke-ui` job |
| 依赖 | 从 `package.json` 移除 `playwright` |
| 文档 | 更新 `README.md`、`docs/api-design/05-cli-commands.md` |

变更后，UI 覆盖仍由 HTTP 级 `test/ui.test.ts` 与 `ao ui --once` 健康检查承担；浏览器级交互回归（自动刷新、筛选器、空状态等）不再自动化。
