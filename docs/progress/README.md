# Progress

This directory records current live status. It is not a plan-only area; update it when work starts, verification runs, contracts change, or blockers appear.

Files:

- `milestones.md`: milestone-level readiness.
- `tasks.md`: executable task queue.
- `contract-checklist.md`: contract status and freeze tracking.
- `test-acceptance-log.md`: exact verification commands and outcomes.
- `blockers.md`: active and resolved blockers.
